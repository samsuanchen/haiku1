# haiku1.html 雲端方寸彩繪 範例集 2018/06/24 陳爽

這是一個 Forth Haiku 雲端方寸彩繪 Script 文本的 IDE  測試或開發開發平台

註0 反斜線空格之後 直到列尾的文字都忽略 作註解
    左圓刮號空格之後 到 對應 右圓刮號 之間文字也都忽略 作註解

註1 最好用 google chrome 瀏覽器 開啟 此網頁

註2 其他範例可參閱 html/PublishedHaiku1.html 與 html/PublishedHaiku2.html

註3 更多範例可參閱 Brad Nelson 的 https://forthsalon.appspot.com/ 網頁

註4 列首為 "  ( \" 的下一列到列首為 "\ ) \" 的上一列 都是 可執行 範例
    手動將 "  ( \" 變為 "\ ( \", 第一個空格變為反斜線, 即 開啟 範例, 自動顯示 圖形
    改其中 數字或指令 即顯示 更動效果, 所以最好參照 註6 複製 範例 以副本進行測試
    將 "\ ( \" 變為 "  ( \", 第一個反斜線變為空格, 即 關閉 範例, 停止顯示 圖形

註5 通常不能同時開啟幾個範例, 手動開啟一個範例後, 最好先手動關閉, 然後開啟另外的範例

註6 要測試範例, 最好先複製範例成副本, 然後開啟這副本, 改其中數字或指令 看更動效果
